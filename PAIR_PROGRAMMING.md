# Pair Programming

## Estratégia

O desenvolvimento das funcionalidades será realizado utilizando a prática de pair programming de forma adaptada.

Quando aplicado em dupla:
- Um desenvolvedor atua como driver (codifica)
- O outro atua como navigator (revisa e orienta)
- Troca de papéis a cada sessão

## Ferramentas

- VS Code Live Share
- Discord para comunicação
- GitHub para versionamento

## Benefícios

- Melhor qualidade de código
- Menos erros
- Maior aprendizado colaborativo
- Revisão contínua durante o desenvolvimento
