# Histórias de Usuário

## História 1: Sistema de Votação

Como usuário do fórum,  
eu quero votar em perguntas (upvote/downvote),  
para destacar perguntas mais úteis para a comunidade.

### Critérios de Aceitação:
- O sistema deve permitir upvote e downvote em perguntas
- O usuário não pode votar mais de uma vez na mesma pergunta
- O voto deve ser registrado corretamente no sistema
- O contador de votos deve ser atualizado


## História 2: Busca de Perguntas

Como usuário do fórum,  
eu quero buscar perguntas por palavra-chave,  
para encontrar informações relevantes rapidamente.

### Critérios de Aceitação:
- O sistema deve permitir busca por texto
- Resultados devem filtrar perguntas corretamente
- A busca deve considerar título e conteúdo
- Deve retornar resultados relevantes


## História 3: Categorização por Tags

Como usuário do fórum,  
eu quero ver perguntas organizadas por tags,  
para facilitar a navegação por temas.

### Critérios de Aceitação:
- Perguntas devem ter tags associadas
- O usuário pode visualizar perguntas por categoria
- As tags devem ser exibidas claramente
- Deve ser possível filtrar por múltiplas tags


## Priorização

1. Busca de perguntas (mais usada e essencial)
2. Sistema de votação (engajamento da comunidade)
3. Tags/categorias (organização do conteúdo)
