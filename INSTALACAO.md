# Instalação do Projeto ESM Forum

## Pré-requisitos
- Node.js versão 18 instalado 
- Git instalado


## Clonagem dos repositórios


git clone https://github.com/lsduartelsdt/fgv-es1-projeto-final-parte1-lsduartelsdt.git
git clone https://github.com/lsduartelsdt/esmforum.git
git clone https://github.com/lsduartelsdt/esmforum-react.git

## Backend rodando na porta 5000

```bash
cd esmforum
npm ci
npm start
```

## Frontend

```bash
cd esmforum-react
npm ci
npm start
```

Acesso em: http://localhost:3000
